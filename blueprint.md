# Project Blueprint

## Overview

This is a Flutter application that showcases a modern and visually appealing UI design. The app consists of three screens, each with a unique but consistent design language. The design incorporates gradient backgrounds, card-based layouts, and modern typography to create an engaging user experience.

## Style and Design

- **Color Palette:** The app uses a vibrant and modern color palette, with each screen featuring a distinct color scheme based on shades of blue, purple, and teal.
- **Typography:** The typography is clean and readable, with a clear visual hierarchy. The font sizes and weights are used effectively to distinguish between headings, subheadings, and body text.
- **Layout:** The layout is based on a card-based design, with the main content of each screen enclosed in a `Card` widget. This creates a sense of depth and visual separation from the background.
- **Buttons:** The buttons are styled with rounded corners and a prominent appearance, making them interactive and user-friendly.

## Features

- **Three Screens:** The app consists of three screens: `InicioScreen`, `Pantalla2Screen`, and `Pantalla3Screen`.
- **Navigation:** The app uses a simple navigation system that allows users to move between the three screens.
- **Modern UI:** The app features a modern and visually appealing UI design that is both engaging and easy to use.

## Current Task: Initial Setup and Design

- **Project Initialization:** The project has been initialized as a standard Flutter application.
- **UI Design:** The UI of all three screens has been designed and implemented with a focus on modern aesthetics and a consistent design language.
- **Testing:** The initial tests have been set up and are passing, ensuring the basic functionality of the app is working correctly.
